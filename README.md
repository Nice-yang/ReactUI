# ReactUI

ReactUI is an interactive React component builder which modifies your original code for you.

![composingComponents](docs/images/composingComponents.gif)

## Keyboard Shortcuts

- Copy: CMD + C
- Paste: CMD + V
- Go to source: CMD + click
- Delete: CMD + Delete
- Save style: CMD + S
- Insert Child Div: CMD + Enter
- Goto Code: CMD + click
